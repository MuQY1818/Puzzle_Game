# 🧩 超级智能拼图游戏 🧠

## 🌟 特性

- 🖼️ 动态加载图片，创造独特拼图体验
- 🔄 多种难度级别，挑战自我
- ⏱️ 挑战模式，与时间赛跑
- 🤖 内置求解器，一键解题
- 🎨 美观的用户界面，流畅的游戏体验

## 🚀 技术亮点

- 💡 采用 A* 算法实现高效解题
- 🧠 启发式函数优化，加速搜索过程
- 🔍 线性冲突检测，提高解题精度
- 🔢 逆序数判定，确保拼图可解性
- 🖥️ Swing GUI，实现流畅交互

## 🛠️ 如何运行

1. 确保您的系统安装了 Java 运行环境 (JRE)
2. 下载最新的发布版本 `PuzzleGame.jar`
3. 双击 JAR 文件或在命令行中运行：
   ```
   java -jar PuzzleGame.jar
   ```

## 🎮 游戏玩法

1. 点击 "随机切换图片" 选择您喜欢的图片
2. 使用 "改变难度" 按钮调整游戏难度
3. 点击拼图块移动，或拖拽拼图块交换位置
4. 挑战自己，看看能否在最短时间内完成拼图！
5. 遇到困难？使用 "一键解题" 功能，观察 AI 如何解决难题
